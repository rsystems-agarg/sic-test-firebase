Configuring an LQ Profile

To Configure an LQ Profile:

  1. Click on the Settings icon, The Settings dialog screen appears.

  2. Ensure that the Profiles tab is selected:

  1. Edit the settings as required by the host intercom system.

  2. Select a role, and then select Done.

Note: The role can be fixed by the LQ CCM interface. In this case, you will not be able to select a role as it will be allocated automatically by the host intercom system.

Once you have accessed the system, a user selectable role is indicated by a star in the Station-IC keysets screen:

Note: If the role for your intercom client has been configured as fixed in the LQ CCM, you cannot select a different role. If you try to select a different role, you will see an exclamation mark in a triangle next to the role selection drop-down. This indicates that the role you have tried to select will not be applied to Station-IC.
